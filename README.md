# A-Z 英文學習 ABC Learning

一個給小朋友的互動英文學習網頁：點單字聽發音，涵蓋 7 大主題（動物、食物、顏色形狀、自然、運動、交通職業、日常用品），以及基礎字母 A-Z 總覽。

## 特色

- **點擊發音**：使用瀏覽器內建語音合成 (Web Speech API)，可選聲音與語速。
- **學習進度**：自動記錄學過的單字（存於瀏覽器 localStorage），顯示總進度條與 ✓ 標記。
- **自訂代表字**：每個字母可用 ⭐ 選擇要當「代表字」的單字，並同步到總覽頁。
- **單一檔案**：整個 App 就是一個 `abc_learning.html`，免安裝、可離線、手機直接開。

## 使用方式

直接用瀏覽器開啟 `abc_learning.html` 即可，不需要伺服器或任何建置步驟。

## 技術

- 純 HTML / CSS / JavaScript，無外部相依套件。
- 進度與代表字設定儲存在使用者裝置的瀏覽器內（localStorage），不會上傳。
