# 日常用品 A-Z · Gemini 生圖 Prompt

風格對照 `images/basketball.jpg` / `images/apple.jpg`:淺奶油底、深色描邊的扁平卡通 illustration,置中主體,右下角小星星裝飾。

檔名規則:`images/<小寫單字>.jpg`,空格與連字號改底線(例:`Alarm Clock` → `alarm_clock.jpg`)。

## 風格共通前綴(每個 prompt 前面貼這段)

